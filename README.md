# Instapro

MVP социальной сети для обмена фотографиями

## Первоначальная оценка

10 часов

## Фактически затраченное время

12 часов

## Функциональность приложения
Неавторизованный пользователь может:
просматривать общую ленту постов;
переходить на страницы других пользователей;
перейти на страницу входа и регистрации.

Авторизованный пользователь может:
зарегистрироваться и войти в аккаунт;
выйти из аккаунта;
просматривать ленту всех постов;
просматривать посты конкретного пользователя;
добавлять новые посты (описание и изображение);
ставить и убирать лайки;
видеть количество лайков у постов;
видеть дату и время публикации поста.
